# Nuvexa by GENIQX

This is the full production-ready deployment for the **Nuvexa** medical AI consent, diagnostics, and robotics system.

## 🧠 Features
- AR/VR/IR body scanning
- Real-time AI triage and surgical overlays
- CPD training + simulation engine
- Secure consent logging + video sessions
- AI personas + multilingual UI
- Admin control, export, diagnostics, telemetry

## 📦 Stack
- Frontend: Next.js + Tailwind
- Backend: Flask (Python)
- CI/CD: GitHub Actions + Render
- AI: GPT + GENIQX Core

## 🛠 Setup
1. Copy `.env.example` → `.env` and fill secrets
2. Push to GitHub
3. Connect to [Render.com](https://render.com/)
4. Auto-deploy via `render.yaml`
